# Changelog

## 1.0.0
- Initial release
- Daily rotating bounty board with 6 bounties per day (2 gather, 2 kill, 2 miniboss)
- 150+ bounty pool spanning all biomes from Meadows to Mistlands
- Miniboss bounties spawn starred creatures with boss health bars and minimap area markers
- Gather bounties track pickups in real-time and require item turn-in on completion
- Kill bounties track creature kills with level-matching for starred spawns
- Boss progression gates unlock higher-tier bounties as you defeat bosses
- Seeded daily rotation ensures all players see the same bounties each day
- Bounty rewards deposited directly to Haldor's Bank
- Abandon system lets you drop unwanted active bounties
- Live countdown timer shows time until next bounty reset
- Color-coded bounty list by difficulty tier
- Full integration with HaldorOverhaul trader UI as a 4th tab
